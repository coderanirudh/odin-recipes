# Odin-recipes
This is the first project in the foundations course in The Odin Project.

In this Project I added three recipes in the homepage, when clicked they will take us to the respective recipe page.

In the recipe page we can see the recipes name, image, desciption, ingredients needed and the instruction on how to make it.

I also added an anchor tag with "Home" to come to the homwpage.

In this project I showcased my basic HTML concepts like adding images, links, ordered and unordered lists, heading tags and paragraph tahs etc.,

I also learned how to search for images that are okay to use in our websites using google's tools.
